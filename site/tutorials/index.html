Tutorials

The homepage has menu, inline links, regular paragraph text, block level code, small text. So this page has the other content types from the Google Doc.

Highlight text

The longer the highlighted text, the less intense the highlighted color should be. Right now this is 0.15 alpha of the mark color (which is as low as it should go, and .35 is about the max). Download the p5.js file and create a new index.html file and a new sketch.js file. You want to link to p5.js and your sketch.js file in the head of the html file. Make sure all three files are in the same folder. You can use any code editor you like, we recommend Sublime Text.

Inline Code

The home page has block level code examples. Let's try a little inline code in a paragraph such as this one. We will make the amount of code small to see how will this works. So, if (x = 1) and then you want to make sure that if you are going to ask the browser to do something you might want to also give it another option, or else.

Alternatively, you can download the whole project and copy the examples/empty-example folder to any location you like and edit the sketch.js file.
