What is p5.js?

p5.js is a JavaScript creative coding toolkit which lets you create generative and interactive art to run in any modern web browser. It is based on Processing and follows many of the same principles, while extending them into the domain of the web.

p5.js is not to be confused with Processing.js, which is a translation engine that lets you run programs written in Processing in a web browser. p5.js is not a translation engine, but a language itself.

2. More advanced drawing

A variable is a way to store data to be reused, modified, accessed.
They are like variables you might remember from algebra, but can be numbers, string (words), and more.
  1. Go to Variable Example
  2. Fork it!
  3. Change the variables around...
A for loop is a way to repeat the same command a multiple times.
  1. Go to For Loop Example
  2. Fork it!
  3. Change the loop to work 100, 200 times
A conditional is like a question. It lets you decide which command to execute based on the truthiness of something.
If (I am hungry), I will eat a hot dog, else, I will drink water.
  1. Go to Conditional Example 1 or Conditional Example 2
  2. Fork it!
  3. Change conditional to do the opposite thing

3. Interaction

Mouse variables give you information about the state of the mouse.
mouseX and mouseY hold the x and y coordinates of the mouse.
  1. Go to Mouse Variables Example
  2. Fork it!
  3. Make an object follow the mouse, like a custom pointer
  4. Change the sketch so there are three horizontal regions instead of two
A mouse event is a command that is automatically called whenever your mouse is clicked or moved.
Different mouse events include mousePressed, mouseClicked, etc.
  1. Go to Mouse Events Example
  2. Fork it!
  3. Add commands to change the color whenever the mouse is pressed
A keyboard event is a command that is automatically called whenever your keyboard is pressed.
  1. Go to Keyboard Events Example
  2. Fork it!
  3. Add commands to draw new lines whenever a key is pressed
A touch event is like a mouse event, but for touches on your touchscreen enabled devices!
Different touch events include touchStarted, touchEnded, etc.
  1. Go to Touch Events Example
  2. Fork it!
  3. Add commands to draw wherever the screen is touched
